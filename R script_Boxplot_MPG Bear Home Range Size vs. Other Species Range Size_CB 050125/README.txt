Title: MPG Bear Home Range Size vs Other Species Home Range Size Boxplot
Author: Carly Brooks
Date: May 1 2025
Contact: cbrooks@mpgranch.com

Description: 
________________

This folder contains an R script file ('MPG Bear Home Range Size vs Other Species Range Size.R') used to generate a boxplot of varying home range sizes for different animals along a social-solitary spectrum

Files Included:
________________

- 'MPG Bear Home Range Size vs Other Species Range Size.R' — Main R markdown file with all code, package loading, annotations, and results. 
- README.txt — This file.
— 1_MPG Home Range Sizes.xlsx - Master doc used to make 'MFRange'. Contains all the annual home range sizes (km2) for collared bears.
— Female_HRsize.xlsx - Contains averaged out home range values for all collared MPG female bears, and other female species. E.g. if ZZ had 4 home range sizes for years 2020-2023, I took the average of all of them.
— Male_HRsize.xlsx - Contains averaged out home range values for all collared MPG male bears, and other male species. E.g. if ZZ had 4 home range sizes for years 2020-2023, I took the average of all of them.
— BlackBear_RangeMap_LitReview_CB.ppt - Literature review document regarding various black bear studies
— RangeMap_LitReview_CB.ppt - Literature review document regarding many different species, used to generate home range size values in the Female and male HRsize sheets.
