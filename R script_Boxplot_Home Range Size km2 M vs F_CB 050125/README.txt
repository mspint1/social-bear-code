Title: MPG Bear Home Range Size Boxplot (km2) Males vs Females
Author: Carly Brooks
Date: May 1 2025
Contact: cbrooks@mpgranch.com

Description: 
________________

This folder contains an R script ('M v F Home Range Size km2.R') to create a boxplot of home range sizes for MPG bears, split into males and females. 

Files Included:
________________

- M v F Home Range Size km2.R — Main R file with all code, package loading, annotations, and results. 
- README.txt — This file.
- 1_MPG Home Range Sizes.xlsx - Master doc used to make 'MFRange'. Contains all the annual home range sizes (km2) for collared bears.
- MFRange.xlsx - Doc of male and female home range sizes to load into R for the boxplot. Contains averaged out home range values for all collared male and female indivdiuals. E.g. if ZZ had 4 home range sizes for years 2020-2023, I took the average of all of them.
