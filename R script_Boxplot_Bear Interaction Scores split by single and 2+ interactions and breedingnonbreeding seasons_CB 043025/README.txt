Title: MPG Bear Interaction Scores, Single and 2 Plus Split, Breeding/Nonbreeding Split
Author: Carly Brooks

Description: 
________________

This folder contains an R script ('InterationScores_Single2Plus_BreedingNonBreeding.R') to create a boxplot of interaction scores split by single/2+ interactions and breeding/nonbreeding seasons. 

Files Included:
________________

- InterationScores_Single2Plus_BreedingNonBreeding.R — Main R file with all code, package loading, annotations, and results. 
- README.txt — This file.
- Boxplot_breednonbreed_041725.xlsx — Main Excel file loaded into R to create boxplot. Contains 12 columns for MM, FF, MF breeding/non breeding 1 and 2+ pts.
- BreedNonBreed_2020_2024_InteractionScores_Summarized.xlsx — Master Interaction Score Excel file where all bear pair scores are outlined.
