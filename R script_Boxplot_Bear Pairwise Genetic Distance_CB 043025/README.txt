Title: MPG Bear Genetic Distance Values, MM vs FF pairs comparisons
Author: Carly Brooks


Description: 
________________

This folder contains an R script ('GD.R') used to create Boxplots of pairwise genetic distance values for MM and FF pairs of American black bear pairs from 2020 to 2024.  

Files Included:
________________

- GD.R — Main R file with all code, package loading, annotations, and results. 
- GD_only study bears has the genetic distance for every bear in the study.
- README.txt — This file.
— GD.xlsx — Excel doc with all pairwise genetic distance values
— 3 YEAR OLD BEARS.docx — List of < 3 year old bears to exclude
— MALES BORN ON THE RANCH.docx — List of Male bears born on the ranch to exclude
